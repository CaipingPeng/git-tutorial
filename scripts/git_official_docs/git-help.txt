[p] git-help - Display help information about Git

[pre] git help[-a|--all] [--[no-]verbose] [--[no-]external-commands] [--[no-]aliases]git help[[-i|--info] [-m|--man] [-w|--web]] [<command>|<doc>]git help[-g|--guides]git help[-c|--config]git help[--user-interfaces]git help[--developer-interfaces]

[h2] DESCRIPTION

[p] With no options and no<command>or<doc>given, the synopsis of thegitcommand and a list of the most commonly used Git commands are printed
on the standard output.

[p] If the option--allor-ais given, all available commands are
printed on the standard output.

[p] If the option--guidesor-gis given, a list of the
Git concept guides is also printed on the standard output.

[p] If a command or other documentation is given, the relevant manual page
will be brought up. Themanprogram is used by default for this
purpose, but this can be overridden by other options or configuration
variables.

[p] If an alias is given, git shows the definition of the alias on
standard output. To get the manual page for the aliased command, usegit<command>--help.

[p] Note thatgit--help...is identical togithelp...because the
former is internally converted into the latter.

[p] To display thegit[1]man page, usegithelpgit.

[p] This page can be displayed withgit help helporgithelp--help.

[p] Print all the available commands on the standard output.

[p] When used with--all, exclude the listing of external "git-*"
commands found in the$PATH.

[p] When used with--all, exclude the listing of configured
aliases.

[p] When used with--all, print description for all recognized
commands. This is the default.

[p] List all available configuration variables. This is a short
summary of the list ingit-config[1].

[p] Print a list of the Git concept guides on the standard output.

[p] Print a list of the repository, command and file interfaces
documentation on the standard output.

[p] In-repository file interfaces such as.git/info/excludeare
documented here (seegitrepository-layout[5]), as well as
in-tree configuration such as.mailmap(seegitmailmap[5]).

variable; they makegit helpbehave as their corresponding command-
