[p] git-difftool - Show changes using common diff tools

[pre] git difftool[<options>] [<commit>[<commit>]] [--] [<path>…​]

[h2] DESCRIPTION

[p] git difftoolis a Git command that allows you to compare and edit files
between revisions using common diff tools.git difftoolis a frontend
togit diffand accepts the same options and arguments. Seegit-diff[1].

[p] Copy the modified files to a temporary location and perform
a directory diff on them. This mode never prompts before
launching the diff tool.

[p] Do not prompt before launching a diff tool.

[p] Prompt before each invocation of the diff tool.
This is the default behaviour; the option is provided to
override any configuration settings.

[p] Start showing the diff for the given path,
the paths before it will move to the end and output.

[p] Start showing the diff for the given path, skipping all
the paths before it.

[p] Use the diff tool specified by <tool>.  Valid values include
emerge, kompare, meld, and vimdiff. Rungitdifftool--tool-helpfor the list of valid <tool> settings.

[code] --tool-help

[p] If a diff tool is not specified,git difftoolwill use the configuration variablediff.tool.  If the
configuration variablediff.toolis not set,git difftoolwill pick a suitable default.

[p] You can explicitly provide a full path to the tool by setting the
configuration variabledifftool.<tool>.path. For example, you
can configure the absolute path to kdiff3 by settingdifftool.kdiff3.path. Otherwise,git difftoolassumes the
tool is available in PATH.
