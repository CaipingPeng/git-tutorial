# Git 基础教程 - 导航

欢迎来到 Git 基础教程!这套教程专注于日常开发中最常用的 Git 操作,覆盖 95% 的使用场景。

## 🎯 学习目标

完成这套教程后,你将能够:
- ✅ 独立管理自己的代码版本
- ✅ 与团队协作开发项目
- ✅ 使用 GitHub 进行开源贡献
- ✅ 解决常见的 Git 问题

## 📚 教程章节

### 第一部分:入门 (30分钟)

1. **[Git 是什么](01-introduction.md)** (5分钟)
   - 为什么需要版本控制
   - Git 的核心概念
   - Git vs 其他工具

2. **[安装与配置](02-installation.md)** (10分钟)
   - 安装 Git
   - 基本配置
   - 第一个仓库

3. **[日常工作流](03-daily-workflow.md)** ⭐ (15分钟)
   - 4个核心命令
   - 完整的工作循环
   - 查看历史和状态

### 第二部分:协作 (45分钟)

4. **[分支基础](04-branching.md)** (15分钟)
   - 创建和切换分支
   - 合并分支
   - 分支管理

5. **[远程协作](05-remote-collaboration.md)** (15分钟)
   - 克隆仓库
   - 推送和拉取
   - 多人协作

6. **[冲突解决](06-conflict-resolution.md)** (15分钟)
   - 理解冲突
   - 解决冲突
   - 使用工具

### 第三部分:实战 (30分钟)

7. **[撤销错误](07-undo-mistakes.md)** (15分钟)
   - 撤销工作目录改动
   - 撤销暂存
   - 回退提交

8. **[GitHub 工作流](08-github-workflow.md)** (15分钟)
   - Fork 和 PR
   - Issue 管理
   - 开源贡献
